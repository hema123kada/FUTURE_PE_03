## 📝 Prompt 2 — SEO Blog Outline
**🤖 Tool:** Claude AI
**🎯 Goal:** Generate a detailed SEO blog outline with
H1-H3 structure for the pillar page topic

**📝 Prompt:**
You are an SEO content writer specializing in IT services.

Create a detailed SEO blog outline for this topic:
"Complete Guide to Website Development for Small
Businesses in Hyderabad"

The outline must include:
1. SEO Title (under 60 characters)
2. Meta Description (under 155 characters)
3. H1 Heading (main title)
4. Introduction paragraph outline (3-4 points)
5. H2 Sections (minimum 5 sections)
6. H3 Subsections (2-3 under each H2)
7. FAQ Section (5 questions)
8. Conclusion outline
9. Internal linking opportunities
10. Target keywords to include naturally

Target keyword: "website development Hyderabad"
Word count target: 1500-2000 words
Tone: Helpful, professional, and easy to understand

**✅ Why This Prompt Works:**
- 📋 Full H1-H3 structure ensures proper SEO formatting
- 🎯 Target keyword instruction improves ranking potential
- ❓ FAQ section captures featured snippet opportunities
- 🔗 Internal linking improves overall site SEO health

## ✍️ Prompt 3 — Full SEO Blog Post
**🤖 Tool:** Claude AI
**🎯 Goal:** Generate a complete long-form SEO blog post
optimized for search intent and local keywords

**📝 Prompt:**
You are an expert SEO content writer.

Write a complete SEO-optimized blog post for
"TechNest Solutions", an IT company in Hyderabad.

Blog Topic:
"Complete Guide to Website Development for Small
Businesses in Hyderabad"

Requirements:
- Word count: 1500-2000 words
- Target keyword: "website development Hyderabad"
  (use naturally 6-8 times)
- LSI keywords: include naturally throughout
  (web design, mobile-friendly, business website,
  Hyderabad IT company, affordable website)
- Structure: H1 → H2 → H3 headings
- Include: Introduction, main sections, FAQ, Conclusion
- Tone: Helpful, professional, easy to read
- End with strong CTA to contact TechNest Solutions
- Add internal link placeholders where relevant

Make content genuinely helpful and business-focused.
Not keyword stuffed — natural and readable.

**✅ Why This Prompt Works:**
- 📏 Word count target ensures long-form ranking content
- 🔄 LSI keyword instruction prevents keyword stuffing
- 📋 Clear structure ensures proper SEO formatting
- 💡 Helpful content instruction improves user experience
- 👉 CTA instruction ensures business conversion focus

## 🕸️ Prompt 5 — Content Cluster & Internal Linking
**🤖 Tool:** Claude AI
**🎯 Goal:** Generate internal linking strategy and
content cluster ideas for building topical authority

**📝 Prompt:**
You are an SEO strategist specializing in content clusters.

Create a complete content cluster strategy for
"TechNest Solutions" IT company website in Hyderabad.

Include:
1. Pillar Page: "IT Services in Hyderabad" (main hub)
2. 5 Cluster Blog Posts supporting the pillar:
   - Topic title for each
   - Target keyword for each
   - 2-line description of what each covers
3. Internal linking map:
   - Which cluster links to pillar
   - Which clusters link to each other
4. Suggested URL slugs for each page
5. Content publishing order recommendation

Goal: Build topical authority for IT services in
Hyderabad and rank on page 1 of Google.

**✅ Why This Prompt Works:**
- 🕸️ Content cluster structure builds topical authority
- 🔗 Internal linking map improves site architecture
- 📅 Publishing order ensures strategic content rollout
- 🎯 Topical authority focus improves overall rankings
